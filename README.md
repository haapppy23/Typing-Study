# 📝 자격증 필기 합격 프리패스 - 무한 빡공 암기장 (Cert-Master)

문제은행식 자격증(리눅스마스터, 정보처리기사, 컴활 등) 필기시험을 완벽하게 부수기 위해 만든 **파이썬 기반 타이핑 암기장**입니다. 
눈으로만 대충 보고 '답 번호'만 외워버리는 꼼수를 원천 차단하고, 직접 키보드를 두드리며 **손가락 근육에 정답을 각인시키는 실전형 학습 도구**입니다.

## ✨ 주요 기능 (Features)

* 🔄 **무한 확장성 (JSON 연동):** 하드코딩된 문제? NO. `quiz_data.json` 파일만 교체하면 어떤 자격증이든 1초 만에 전용 암기장으로 변신합니다. (자동으로 30문제씩 Set 분할)
* ✍️ **자유 빡공 모드 (1차 학습):** 지문을 보며 생각나는 키워드를 연습장 칸에 마음대로 끄적일 수 있습니다. 오타, 띄어쓰기 신경 쓰지 마세요. 밑에 있는 '정답 칸'에만 정확한 답을 치면 쿨하게 패스!
* 🔥 **실전 테스트 모드 (랜덤 셔플):** 앞뒤 문제 순서로 답을 유추하는 꼼수를 막기 위해, 테스트 진입 시 문제 순서를 무자비하게 섞어버립니다.
* 🎵 **타격감 200% 사운드 이펙트:** 타이핑할 때마다 경쾌한 기계식 키보드 소리가 출력되어 학습 집중력을 극대화합니다.
* 📝 **즉석 오답노트 (메모장):** 공부하다 헷갈리는 개념이 생기면 버튼 한 번으로 윈도우 메모장을 호출하여 바로바로 기록할 수 있습니다.
